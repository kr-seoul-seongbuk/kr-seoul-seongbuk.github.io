# 성북구 · 서울특별시

> 성북구 지역 정보 랜딩 사이트

## 🌐 랜딩 페이지

### **https://kr-seoul-seongbuk.github.io/**

성북구는 한양도성 성곽의 북쪽 자락에 기대어 형성된 유서 깊은 문화·학문의 고장입니다. 구의 이름은 조선 영조 때 어영청 소속 북둔(北屯)이 도성 북쪽에 자리했던 데서 비롯되었으며, 성북동이라는 지명은 1894년 갑오개혁 무렵 문헌에 처음 등장합니다. 간송미술관과 심우장, 길상사, 최순우 옛집이 한옥 골목 사이에 이웃해 있어 서울에서 손꼽히는 근대문화 예술 밀집지로 통합니다. 고려대·국민대·성신여대·한성대를 비롯한 여러 대학이 밀집해 젊은 학구적 분위기가 도시 전역에 스며 있습니다. 북한산 자락의 정릉계곡과 개운산·오패산의 녹지가 도심 주거지와 맞닿아 자연과 생활이 가까운 곳입니다.
